# EduFlow Docs

MVP веб-системы электронного документооборота учебного заведения.

## Что реализовано

- регистрация сотрудников с подтверждением администратором или директором;
- вход по JWT и разграничение прав по ролям;
- личный кабинет с виджетами, уведомлениями и быстрым доступом к модулям;
- внутренние сообщения между сотрудниками и отправка по подразделениям;
- создание и согласование заявлений;
- загрузка и просмотр документов;
- отчёты и журнал действий для административных ролей.

## Стек

- `frontend`: React + Vite
- `backend`: Node.js + Express
- хранилище MVP: JSON-файл `backend/data/db.json`

## Запуск

### Backend

```bash
cd backend
npm install
npm run dev
```

API будет доступен на `http://localhost:4000`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Клиент будет доступен на `http://localhost:5173`.

## Тестовые аккаунты

- `admin / admin123`
- `director / director123`
- `teacher / teacher123`
- `academic / academic123`
- `hr / hr123456`
- `accountant / account123`

## Примечания

- Для документов используется локальная папка `backend/uploads`.
- Для production-версии стоит заменить JSON-хранилище на PostgreSQL или MySQL.
- Восстановление пароля в текущем MVP оформлено как уведомление администратору.
